Exercises¶

  1. Create a list containing 100 random integers between 0 and 1000 (use iteration, append, and the random module). Write a function called average that will take the list as a parameter and return the average.

  2. Write a Python function that will take a list of 100 random integers between 0 and 1000 (you can use iteration, append, and the random module to create a test list) and return the maximum value. (Note: there is a built-in function named max but do not use it for this exercise.)

  3. Write a function sum_of_squares(xs) that computes the sum of the squares of the numbers in the list xs. For example, sum_of_squares([2, 3, 4]) should return 4+9+16 which is 29:

  4. Sum up all the negative numbers in a list.

  5. Count how many words in a list have length 5.

  6. Count how many words occur in a list up to and including the first occurrence of the word “sam”.

  7. Although Python provides us with many list methods, it is good practice and very instructive to think about how they are implemented. Implement your own Python functions that works like the following built-in ones:

    1. count
    2. in
    3. reverse
    4. index
    5. insert

  8. Write a function replace(s, old, new) that replaces all occurences of old with new in a string s:

    test(replace('Mississippi', 'i', 'I'), 'MIssIssIppI')

s = 'I love spom! Spom is my favorite food. Spom, spom, spom, yum!'
test(replace(s, 'om', 'am'),
       'I love spam! Spam is my favorite food. Spam, spam, spam, yum!')

test(replace(s, 'o', 'a'),
       'I lave spam! Spam is my favarite faad. Spam, spam, spam, yum!')

    Hint: use the split and join methods.
